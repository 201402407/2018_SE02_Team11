# 2018_SE02_Team11
- charset은 EUC-KR
- 모든 UI페이지는 jQuery(자바스크립트 라이브러리)를 포함해야 한다. AJAX 기능을 쉽게 사용하기 위함.
- jQuery는 "/external_lib/jquery-3.3.1.min.js" 에 있다.
- Proc페이지 : UI페이지로부터 입력을 받아 DAO클래스 오퍼레이션을 실행하여 그 결과를 출력하는 페이지.
- UI페이지는 form데이터를 Proc페이지에 submit할 때 AJAX를 이용한다. (즉, submit을 누를 때 브라우저는 Proc페이지로 이동을 하지 않는다.)
- UI페이지의 파일명 규칙: 딱히 없음
- Proc페이지는 디렉토리 "/proc"에 위치한다.
- Proc페이지의 파일명 규칙: 엔티티타입_오퍼레이션이름.jsp (예: account_signup.jsp)
- 디렉토리 document_se02_11 : 요구사항문서부터 명세서DAO까지, 즉 지금까지 작성해온 분석 및 설계문서
- 클래스 Util.OurTimes : 현재 시각을 알고싶을 때. 현재가 수강신청 기간인지 알아야 할 때. 데이터베이스 쿼리로 얻은 날짜데이터를 자바 본연의 날짜데이터로 바꿔야 할 때.....
- [시간 개념]
- 학기중이란? 학기가 시작하는 날(BEGIN_TERM) ~ 학기가 종강하는 날(END_TERM)
- 학기중이 아닐 때 "현재 학기"는 없음. (예: 현재 = 2015년 여름방학 -> "당학기" = 없음)
- 당학기운용과목조회를 학기중이 아닐때 하려면? 수강신청이 시작하는 날(BEGIN_ADD_ATTENDANCE)부터 할 수 있으며, 이 때 현재학기(당학기)는 "앞으로 다가올 학기"를 가리킨다. (예: 현재 = 2015년 여름방학 -> "(조회하려할 때)당학기" = 2015년 2학기)
- 당학기운용과목조회를 학기중에 하면? 당연히 현재 학기에 맞게 하면 된다.
- 강의계획서 작성은 분반추가 시에만 할 수 있다.
